Remarks

Zawiera implementacje interfejsów IComparable oraz IEquatable w celu porównywania ze sobą użytkowników. W związku z integracją aplikacji z bazą danych, wszelkie dane pobierane są z bazy, a jej budowa odpowiada za brak powtórzeń i sortowanie.
