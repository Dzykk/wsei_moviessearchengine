Remarks

Zawiera implementacje interfejsów IComparable oraz IEquatable w celu porównywania i sortowania filmów. W związku z integracją aplikacji z bazą danych, to ona odpowiada za sprawdzenie, czy nie występują powtórzenia. Porównywanie jest jednak wykorzystywane w celu wyświetlania filmów w kolejności w interfejsie graficznym. Zawiera również przeciążoną metodą ToString;
